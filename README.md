# GitHub Contribution Generator

This script generates between 25-30 commits per run to artificially inflate your GitHub contribution graph. 

## ⚠️ Important Warning

**Using this script violates GitHub's Terms of Service and is strongly discouraged. GitHub has systems to detect artificial activity, and accounts caught using such methods may face penalties including account suspension.**

Genuine contributions to real projects are always more valuable than artificially inflated statistics.

## How to Use

1. Navigate to the repository you want to add commits to
2. Make sure you have initialized a git repository with `git init`
3. Run the script:
   ```
   python github_commit_generator.py
   ```

## Features

- Generates 25-30 commits per execution
- Creates random content to make commits appear legitimate
- Adds random commit messages
- Designed to run in an existing git repository
