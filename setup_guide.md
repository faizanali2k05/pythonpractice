# Daily Coding Practice Tracker

## Overview
This is a legitimate application that tracks your daily coding practice and creates real, meaningful contributions to GitHub.

## How It Works
- Creates a genuine coding practice log
- Tracks your daily programming activities
- Generates real commits with actual progress
- Maintains a streak counter
- Creates a public repository you can showcase

## Setup Instructions

1. **Run the application:**
   ```bash
   python app.py
   ```

2. **Make it public:**
   - Create a new repository on GitHub
   - Push the generated folder to GitHub
   - Set repository to public

3. **Daily usage:**
   - Run `python app.py` every day
   - Each run creates 1 legitimate contribution
   - For 25 contributions, run it 25 separate days
